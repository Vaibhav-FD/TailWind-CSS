MARGINS : TAILWIND PROVIDE SO MANY MARGIN VALUES TO USE TO WE SHOULD WRITE.


m-value --------- [ Margin ]
mt-value --------- [ Margin-Top ]
mb-value --------- [ Margin-Bottom ]
ml-value --------- [ Margin-Left ]
mr-value --------- [ Margin-Right ]
mx-value --------- [ Margin-Inline Or X Axis Horizontal Axis ]
my-value --------- [ Margin-Block Or Y Axis Vertical Axis ]

THIS PARENT BOX HAVE MarginBlock-16
BOX I --- THIS ALL BOXES HAVE Margin-9.
BOX II --- THIS ALL BOXES HAVE Margin-9.
BOX III --- THIS ALL BOXES HAVE Margin-9.

PADDINGS : TAILWIND PROVIDE SO MANY PADDING VALUES TO USE TO WE SHOULD WRITE.


p-value --------- [ Padding ]
pt-value --------- [ Padding-Top ]
pb-value --------- [ Padding-Bottom ]
pl-value --------- [ Padding-Left ]
pr-value --------- [ Padding-Right ]
px-value --------- [ Padding-Inline Or X Axis Horizontal Axis ]
py-value --------- [ Padding-Block Or Y Axis Vertical Axis ]

THIS PARENT BOX HAVE Padding-16
BOX I --- THIS ALL BOXES HAVE Padding-9.
BOX II --- THIS ALL BOXES HAVE Padding-9.
BOX III --- THIS ALL BOXES HAVE Padding-9.
